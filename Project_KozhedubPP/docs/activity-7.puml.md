title Диаграмма деятельности: Просмотр отчета по выпускаемой продукции

|Руководитель|
start
:Открывает модуль отчетности;
:Выбирает "Отчет по выпускаемой продукции";
:Указывает фильтры (интервал времени, цех, категория техники);
:Нажимает кнопку "Рассчитать объемы";

|Система|
:Считывает данные из баз изделий и маршрутов;
:Рассчитывает количество полностью собранных машин за период;
:Группирует объемы выпуска по видам и конкретным моделям;
:Визуализирует аналитическую сводку на экране;
stop
